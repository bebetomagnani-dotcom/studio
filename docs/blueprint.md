# **App Name**: Meta Ads Tax Navigator

## Core Features:

- Tax-Inclusive Budgeting: Calculates effective media budget after 12.15% tax, for CFO perspective.
- Tax-Exclusive Budgeting: Calculates total cost including tax to reach a target media spend, for media buyer perspective.
- Real-time Scenario Simulation: Interactive sliders for CTR & conversion rates to simulate profitability changes.
- Profitability Display: Shows a prominent figure displaying the ROAS and Net Profit of simulated marketing campaign
- Educational Tooltips: Provides context-sensitive explanations of complex metrics and tax implications.

## Style Guidelines:

- Primary color: Electric Indigo (#4F46E5) for action and a vibrant feel.
- Background color: Deep Slate (#0F172A) for a modern dashboard look.
- Alert color: Amber Warning (#F59E0B) to indicate tax and cost considerations.
- Headline font: 'Inter Tight' (sans-serif) in bold weights for conveying financial data.
- Body font: 'Geist Mono' (monospace) for input labels and precise technical elements.
- Split-screen layout: inputs on the left, sticky real-time outputs with prominent display on the right.
- Count-up animation on result changes to give a sense of sophisticated processing.
